# thensor

ESP-IDF (Espressif IoT Development Framework) Thermal & Humidity Sensor

I started this project to learn ESP-IDF and to create a cheap and efficient temperature & humidity sensor for my home assistant instance.

## components:
- [FireBeetle 2 ESP32-C6](https://www.digikey.ca/en/products/detail/dfrobot/DFR1075/24344261)
- [AHT21 Sensor](https://www.digikey.ca/en/products/detail/groundstudio/RNQMCD-AAA/21080873?s=N4IgTCBcDaIEoDkCKBZAwgEQPoEE8gF0BfIA)

## features
- [Matter](https://en.wikipedia.org/wiki/Matter_(standard)) connectivity for ease of setup and efficient data transfer
